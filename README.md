# first-JS-game
it's my first game made with JS, HTML and CSS <br>
it was inspired by the google's dino game, but more simple and (tried to be) cuter <br>
<img width="800" height="500" alt="first screen" src="images/beginning_screen.png">
the main idea of the game is to help a goose who loves flowers to avoing stepping on one of them, so when you touch the flower, you lose :(

## Use instructions
clicking anywhere of the screen, the duck will jump <br>
so the player need to help the goose avoid stepping on the flower <br>
if the goose touches the flower, this screen will be showed <br>
<img width="800" height="500" alt="lose screen" src="images/lose_screen.png">
<br>
there will be a stopwatch and your best score <br>
<img width="800" height="500" alt="timer and best score" src="images/timer_bestscore.png">
